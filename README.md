# Titanic! (Go Rewrite)

This is a work-in-progress rewrite of the [Titanic!](https://github.com/osuTitanic/titanic) services written in python.  
These services include:

- [ ] Anchor (Bancho game server)
- [ ] Deck (Score server / Client API)
- [ ] Stern (Website)
- [ ] Keel (API)
- [x] Jobs (Background Tasks)
- [ ] BanchoBot (Discord Bot)
